Music Player UI

The purpose of this project was to learn the basic WPF and xaml. The outcome was to create a music player UI that was similar to something like the Spotify player. Song and album data was accessed through the Spotify API.

Packages Used:

  • RESTSharp
  • Newtonsoft.Json

Features Learned:

  • INotifyPropertyChanged
  • HttpListener
  • xaml:
  • - MVVM Framework
  • - Custom Fonts
  • - Creating Resource Libraries
  • - Creating style templates
