Stock Price Tracker

Link to GitHub repo is here

I started this project with the idea of creating something like a portofolio that could track your stocks. You would be able to see the price of different stocks and the amount of they have risen and fallen within a certain time period. I used ASP.NET MVC framework to build the webapp and the free AlphaVantage API to get the data. Its not completely done yet, I still have to come back to work on it.

Packages Used:

  • RESTSharp
  • Newtonsoft.Json

Future Work To Come Back To:

  • Add some stats that show the price differential between certain time periods.
  • Add a user profile which you can add your stocks to.
  • Add a system where the user can add the stock they want to check instead of having them stored in the database
